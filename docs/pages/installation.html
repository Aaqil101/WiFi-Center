Installation Guide

Welcome to the installation guide! This guide will help you set up the application on your system. Follow the steps below to install the application successfully.

System Requirements

  • Operating System: Windows 10 or newer
  • Python: Version 3.9 or higher installed and added to your system's PATH.
  • Dependencies: Requires packages listed in requirements.txt, including PyQt6, Cython, and NumPy.

Installation Steps (Running from Source)

  1. Clone the Repository: Open a terminal or command prompt and clone the project repository from GitHub: 
    git clone https://github.com/Aaqil101/WiFi-Center.git
  2. Navigate to Project Directory: Change your current directory to the cloned project folder: 
    cd WiFi-Center
  3. (Optional but Recommended) Create a Virtual Environment: To avoid conflicts with other Python projects, it's best to create and activate a virtual environment: 
    python -m venv .venv
  4. Activate the Virtual Environment: Use the following command to activate the virtual environment: 
    .\.venv\Scripts\activate
  5. Install Requirements: Install all necessary dependencies using pip: 
    pip install -r requirements.txt
  6. Build Cython Extensions: Compile the Cython extensions for better performance. Run the following command: 
    cd .\core\
    python setup.py build_ext --inplace
    Note: This may take a few moments as it compiles the Cython code. Ensure you have a C/C++ compiler installed if you encounter issues during this step (e.g., Microsoft C++ Build Tools on Windows).
  7. Run the Application: Start the main application by running the master.py script: 
    python master.py
