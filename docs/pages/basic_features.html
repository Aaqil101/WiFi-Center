Basic Features

The application includes the following basic features:

  • Network Scanner: Scan and display available WiFi networks.

    The application automatically scans for available Wi-Fi networks and displays them in a table. The table shows the network name and the signal strength.

  • Connection Manager: Connect to and manage WiFi networks.

    To connect to a Wi-Fi network, select the network from the table and enter the password. The application will then connect to the network.

  • Command Bar: Execute commands to manage the application.

    The command bar allows you to enter commands to perform various actions, such as disconnecting from the current network, refreshing the list of networks, shutting down the system, rebooting the system, etc. Type help to see a list of available commands.

  • Help: Access this help guide for assistance.

    This help guide provides detailed instructions on how to use the application.

Command Processor

The CommandProcessor class handles the processing of user input and the execution of commands.

Commands

  • quit, exit, close, terminate: Quits the application.
  • disconnect: Disconnects from the current network.
  • refresh: Refreshes the list of networks.
  • shutdown: Shuts down the system.
  • reboot, restart: Reboots the system.
  • sleep: Puts the system to sleep.
  • hibernate: Hibernates the system.
  • lock, logout: Locks the system or logs out the current user.
  • wifi-manager: Opens the Wi-Fi manager.
  • connect: Connects to a specified network.
  • help: Displays a list of available commands.

Terminal AutoComplete

The TerminalAutoComplete class provides auto-completion functionality for the command bar.

It extends the QLineEdit widget and suggests commands as the user types, Cycles through suggestions using the Tab key.

UI Elements

The application's user interface consists of the following elements:

  • Main Window:

    The main window displays the list of available Wi-Fi networks and the command bar.

  • Network Table:

    The network table displays the list of available Wi-Fi networks and their signal strengths.

  • Command Bar:

    The command bar allows the user to enter commands to perform various actions.

  • Output Box:

    The output box displays messages to the user.
