Frequently Asked Questions (FAQ)

This page contains answers to common questions about the Wi-Fi Center application. If you have a question that is not answered here, please check the Contributions page for information on how to ask for help or report issues.

General Usage

Q: How do I exit the Wi-Fi Center application?

A: You can click the standard 'X' button in the window's title bar, press Alt+F4, or type one of the exit commands (quit, exit, close, terminate) into the command bar and press Enter.

Q: How do I refresh the list of Wi-Fi networks?

A: Type -r | refresh into the command bar and press Enter. The background scanner also updates the list automatically at regular intervals.

Q: How do I connect to a Wi-Fi network?

A: Select the desired network using the -c <SSID> | connect <SSID> command. If the network is secured, a password dialog will appear for you to enter the required credentials.

Q: How do I disconnect from the current Wi-Fi network?

A: Type -d | disconnect into the command bar and press Enter.

Q: Can I connect to hidden networks (networks that don't broadcast their SSID)?

A: Currently, the application primarily lists networks detected during scans. Connecting to hidden networks might require using the native Windows Wi-Fi manager (accessible via the -w | wifi-manager command).

Q: How can I see the password I'm typing?

A: There should be a "Show Password" checkbox or icon next to the password input field that appears when you select a secured network.

Q: Where can I see connection status or error messages?

A: Check the Output Box located directly below the command bar at the bottom of the window.

Background Scanner

Q: What is the icon in my system tray?

A: That's the icon for the background Wi-Fi scanner process (core/wifi_scanner.py). It runs continuously to provide network data to the main application. See the Advanced Topics page for details.

Q: Why does the scanner run separately?

A: Running the scanner as a separate background process ensures that network scanning doesn't freeze the main application's user interface and allows for continuous updates.

Q: Can I stop the background scanner?

A: Yes, right-click the scanner's system tray icon and select "Exit". However, the main Wi-Fi Center application will stop receiving network updates if the scanner is not running.

Q: What information is in the wifi_data.json file?

A: It contains a JSON list of the top 6 networks detected by the background scanner, including their SSID, signal strength, whether login is required, and the last time they were seen. You can view it by right-clicking the scanner's tray icon and selecting "View WiFi Data".

Commands

Q: How do I know which commands are available?

A: You can refer to the Basic Features documentation.

Q: Why do commands like shutdown or reboot require confirmation?

A: These are potentially disruptive actions, so the application asks for confirmation to prevent accidental system shutdowns or restarts.

Other

Q: Where is the settings menu?

A: Currently, Wi-Fi Center does not have a dedicated settings menu. Configuration is primarily handled through files (like stylesheets in the styles/ directory).

Q: How can I contribute to the project?

A: Please refer to the Contributions page for guidelines on how to contribute code, report bugs, or suggest features.
