Introduction

Welcome to the Wi-Fi Center Help Guide! This guide will assist you in using the application. Here you will find detailed instructions on how to navigate and utilize the features of the application.

Wi-Fi Center is a PyQt6 application that allows you to manage your Wi-Fi connections. With Wi-Fi Center, you can:

  • Scan for available Wi-Fi networks.
  • Connect to Wi-Fi networks.
  • Disconnect from Wi-Fi networks.

How To Use

Follow these steps to get started with the application:

  1. Open the application.
  2. Use -c <SSID> |connect <SSID> to connect to the network. If the network is secured, a password dialog will appear for you to enter the required credentials.
  3. Take a look at the commands section in the basic features page to see the available commands you can use in the command bar.
  4. Note: You can combine multiple commands using the && operator. For example, to refresh the network list and then connect to a network, you can use: refresh && connect "My Network".
