Troubleshooting Guide

If you encounter problems while installing or using Wi-Fi Center, consult the common issues and solutions below.

It covers the following topics:

  • Installation Issues
  • Background Wi-Fi Scanner
  • Connection Issues
  • General Issues
  • Getting More Help

Installation Issues

  • Issue: Error during pip install -r requirements.txt, especially related to Cython or NumPy.

    Solution: Ensure you have Python 3.9+ and pip installed correctly. You might also need a C/C++ compiler for Cython. On Windows, install the Microsoft C++ Build Tools (select "Desktop development with C++" during installation). Try running the pip install command again in an administrator terminal.

  • Issue: Application fails to start after installation.

    Solution: Double-check that all requirements were installed successfully. Ensure you are running the application from the correct directory (the root `WiFi-Center` folder) using python master.py. Check if you activated your virtual environment (if you created one).

Scanning Issues

  • Issue: No Wi-Fi networks appear in the list, or the list doesn't update.

    Solution:

    1. Ensure your computer's Wi-Fi adapter is enabled.
    2. Run Wi-Fi Center with administrator privileges (right-click the script or shortcut and select "Run as administrator"). Scanning often requires elevated permissions.
    3. Check if the background scanner process (core/wifi_scanner.py) is running. Look for its icon in the system tray. If it's not there, try restarting the main application. If it is there, right-click it, select "Show Console", and check the logs for errors.
    4. Try the -r | refresh command in the command bar.
    5. Ensure no other application is exclusively controlling the Wi-Fi adapter.

  • Issue: The background scanner (system tray icon) is not running or crashes.

    Solution:

    1. Try running python core/wifi_scanner.py directly from the terminal (with administrator privileges) to see if any errors are printed.
    2. Check if another instance is already running (the scanner prevents multiple instances). You might need to use Task Manager to find and terminate existing `python.exe` or `WiFi-Scanner.exe` processes related to this application.
    3. Ensure all dependencies from requirements.txt are installed correctly.

  • Issue: Scan results seem outdated or slow.

    Solution: The background scanner runs at a set interval (default 0.5 seconds). You can force an immediate scan using the -r | refresh command in the main app or the "Force Scan Now" button in the scanner console. Check the scanner console logs for any performance warnings or errors.

Connection Issues

  • Issue: Cannot connect to a network (e.g., "Connection Failed" message).

    Solution:

    1. Double-check that you entered the correct password. Passwords are case-sensitive.
    2. Ensure you have selected the correct network (SSID).
    3. Try connecting using the native Windows Wi-Fi manager (use the -w | wifi-manager command) to see if the issue is specific to Wi-Fi Center.
    4. Run Wi-Fi Center with administrator privileges, as connecting sometimes requires them.
    5. Check the signal strength. Weak signals can cause connection failures.
    6. Restart your Wi-Fi router.

  • Issue: Application freezes when trying to connect.

    Solution: This might indicate an issue with the underlying `pywifi` library or system interaction. Try restarting the application (with administrator privileges). If the problem persists, check the project's GitHub issues page or consider reporting it.

General Issues

  • Issue: Command bar commands don't work (e.g., shutdown, reboot).

    Solution: These commands often require administrator privileges. Ensure you are running Wi-Fi Center as an administrator.

  • Issue: UI elements look strange or misaligned.

    Solution: This could be related to screen scaling settings in Windows or specific theme files. Check the styles/ directory and the OS-specific QSS files (win10.qss, win11.qss). You might need to adjust your display settings.

Getting More Help

If your issue is not listed here:

  • Check the Scanner Console logs for detailed error messages (Right-click tray icon -> Show Console).
  • Consult the FAQ page.
  • Check the GitHub Issues page for existing reports or create a new one.
  • Contact support via the details on the Introduction page.
