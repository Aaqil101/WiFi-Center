Basic Features

The application includes the following basic features:

  • Network Scanner: Scans and displays available Wi-Fi networks.

    When the application starts, it automatically launches a background process called "WiFi Scanner". This process continuously scans for nearby Wi-Fi networks and updates a JSON file with the latest results. The scanned networks are displayed in a table, showing the network name (SSID) and signal strength in real-time. Users can also manually trigger a new scan by using the -r | refresh command for updated results.

  • Connection Manager: Connects to and manages Wi-Fi networks.

    To connect to a network, select it from the table. If the network requires a password, an input field will appear. Enter the password and click the 'Connect' button or press Enter. Connection status (success or failure) will be shown in the Output Box below the command bar. You can also connect using the connect "Network Name" command.

  • Command Bar: Executes commands for various actions.

    Located at the bottom of the window, the command bar allows you to type commands for quick actions like refreshing the network list, disconnecting, managing system power states, or getting help. It features autocompletion to assist you.

  • Help Guide: Accesses this documentation.

    Provides detailed instructions and information about using the application.

  • Command Autocompletion: Suggests matching commands as you type.

    Implemented in core/inline_autocomplete.py, the command bar provides autocompletion. Press the Tab key to complete the suggested command.

Command Processor Details

The application uses a CommandProcessor to interpret and execute commands entered into the command bar.

Available Commands

The following commands are available in the command bar. Type the command and press Enter.

  • quit | exit | close | terminate ➜ Closes the Wi-Fi Center application.
  • -d | disconnect ➜ Disconnects from the currently connected Wi-Fi network.
  • -w | wifi-manager ➜ Opens the native Windows Wi-Fi settings panel.
  • connect <SSID> | Network Name ➜ Attempts to connect to the specified network SSID or name. If the network is secured, a password dialog will appear for you to enter the required credentials.
  • Note: If the SSID contains spaces, enclose it in double quotes. Example: connect "My Network"
  • -r | refresh ➜ Initiates a new scan for available Wi-Fi networks and updates the list.
  • shutdown ➜ Initiates a system shutdown (requires confirmation).
  • reboot | restart ➜ Initiates a system reboot (requires confirmation).
  • sleep ➜ Puts the system into sleep mode.
  • hibernate ➜ Puts the system into hibernation mode (if supported).
  • lock | logout ➜ Locks the workstation or logs out the current user.
  • -h | --help ➜ Displays a summary of available commands in the output box.
  • Note: You can combine multiple commands using the && operator. For example, to refresh the network list and then connect to a network, you can use: refresh && connect "My Network".

Key UI Elements

The application's user interface consists of the following elements:

  • Main Window ➜ The primary application window containing all other elements.
  • Network Table ➜ Displays the list of scanned Wi-Fi networks (SSID, signal strength). Allows selection for connection.
  • Command Bar ➜ The command bar allows users to input commands for various actions, such as connecting to networks or refreshing the network list.
  • Output Box: ➜ Area below the command bar that displays status messages, command output, connection results, and error messages, hidden by default.
  • Password Input ➜ Appears dynamically when connecting to a secured network. Allows users to enter the network password. Includes a "Show Password" checkbox for visibility.
