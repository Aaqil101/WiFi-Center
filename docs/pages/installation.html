Installation Guide

Welcome to the installation guide! This guide will help you set up the application on your system. Follow the steps below to install the application successfully.

System Requirements

  • Windows 10 or newer
  • Python 3.9+ installed
  • Administrator privileges for WiFi operations

Installation Steps

  1. Clone the repository
  2. Navigate to the project directory: cd WiFi-Center
  3. Install requirements: pip install -r requirements.txt
  4. Run master.py to start the application: python master.py
