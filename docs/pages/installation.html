Installation Guide

Welcome to the installation guide! This guide will help you set up the application on your system. Follow the steps below to install the application successfully.

System Requirements

  • Operating System: Windows 10 or newer
  • Python: Version 3.9 or higher installed and added to your system's PATH.
  • Dependencies: Requires packages listed in requirements.txt, including PyQt6, Cython, and NumPy.
  • Privileges: Administrator privileges are required for certain Wi-Fi operations like scanning and connecting/disconnecting networks. The application may prompt for elevation if needed.

Installation Steps (Running from Source)

  1. Clone the Repository: Open a terminal or command prompt and clone the project repository from GitHub: 
    git clone https://github.com/Aaqil101/WiFi-Center.git
  2. Navigate to Project Directory: Change your current directory to the cloned project folder: 
    cd WiFi-Center
  3. (Optional but Recommended) Create a Virtual Environment: To avoid conflicts with other Python projects, it's best to create and activate a virtual environment: 
    python -m venv .venv
# On Windows PowerShell:
.\.venv\Scripts\Activate.ps1
# On Windows Command Prompt (cmd.exe):
.\.venv\Scripts\activate.bat
# On macOS/Linux:
source .venv/bin/activate
  4. Install Requirements: Install all necessary dependencies using pip. This step will also compile the Cython extension (`core/scanner/wifi_scanner_cy.pyx`) needed for efficient network scanning: 
    pip install -r requirements.txt
    Note: This may take a few moments as it downloads packages and compiles the Cython code. Ensure you have a C/C++ compiler installed if you encounter issues during this step (e.g., Microsoft C++ Build Tools on Windows).
  5. Run the Application: Start the main application by running the master.py script: 
    python master.py
    Remember to run this command with administrator privileges if you encounter issues with Wi-Fi scanning or connection management.

Building Standalone Scanner (Optional)

The project includes a separate, optimized Wi-Fi scanner component in the core/scanner/ directory. While the main application uses its compiled Cython version, you can also build a standalone WiFi-Scanner.exe using PyInstaller. This is generally intended for development or distribution purposes.

To build it, navigate to the core/scanner/ directory and run the provided PowerShell script: 

cd core/scanner
powershell -ExecutionPolicy Bypass -File .\build_executable.ps1

This script handles setting up a virtual environment (if needed), running PyInstaller, and placing the final WiFi-Scanner.exe in the core/scanner/ directory.
